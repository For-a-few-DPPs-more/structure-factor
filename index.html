Welcome to structure-factor’s documentation:¶

This is the documentation of the structure-factor project: an open source Python toolbox designed to approximate the structure factor of a stationary point process, test its effective hyperuniformity and identify its class of hyperuniformity.

  • source code on GitHub

  • tutorial Jupyter notebook

Introduction¶

In condensed matter physics, it has been observed for some particle systems that, the variance of the number of points in a large window is lower than expected, a phenomenon called hyperuniformity. To study the hyperuniformity of a given point process, common practice in statistical physics is to estimate a spectral measure called the structure factor, the behavior of which around zero is a indicator of hyperuniformity. The structure factor of a point process is defined via the Fourier transform of its total correlation function; see [Cos21] for more details.
