Hyperuniformity¶

class structure_factor.hyperuniformity.Hyperuniformity(k_norm, sf, std_sf=None)[source]¶

Bases: object

Compute indicator of hyperuniformity of a stationary isotropic (or effectively isotropic) point process \(\mathcal{X} \subset \mathbb{R}^d\), given the evaluation of its structure factor.

This class contains:

  • A method for regularizing the approximated structure factor sample, bin_data(), consisting on dividing the vector of wavelength into sub-intervals and taking the mean and strandard deviation over each sub-interval.

  • Test of effective hyperuniformity, effective_hyperuniformity(), consisting on evaluating the index H of hyperuniformity [KLovriAC+19] and used to study if the sample is effectively hyperuniform.

  • Test of power decay of the structure factor near zero, hyperuniformity_class(), used to determine the class of hyperuniformity of the sample [Cos21].

Typical usage:

1- Estimating the structure factor of a point process by one of the methods of StructureFactor.

2- Regularize the results using bin_data().

3- Testing the effective hyperuniformity using effective_hyperuniformity().

4- If the test effective_hyperuniformity() , approve the effective hyperuniformity hypothesis then use hyperuniformity_class() to study the possible power decay of the structure factor to zero which specify the class of hyperuniformity.

bin_data(**params)[source]¶

Split the vector attribute k_norm into sub-intervals (or bins) and evaluate over each sub-interval the mean and the standard deviation of the corresponding values of the vector attribute sf.

Parameters

params (dict) – parameters associated to _bin_statistics().

Returns

  • self.k_norm: vector of centers of the bins, representing the new vector attribute k_norm.

  • self.sf: corresponding vector of means of the structure factor sf over the bins, representing the new vector attribute sf.

  • self.std_sf: vector of standard deviations, representing the new vector attribute std_sf.

Return type

tuple(np.ndarray, np.ndarray, np.ndarray)

Example

from structure_factor.hyperuniformity import Hyperuniformity

# initialize Hyperuniformity
hyperuniformity_test = Hyperuniformity(k_norm, si)
# regularization of the approximated result
hyperuniformity_test.bin_data(bins=40)
effective_hyperuniformity(k_norm_stop=None, **kwargs)[source]¶

Evaluate the H index of the attribute sf. If \(H<10^{-3}\) the corresponding point process is deemed effectively heyperuniform.

Parameters

k_norm_stop (float, optional) – threshhold on sf used for the linear regression. Defaults to None.

Keyword Arguments

Returns

the value of the index \(H\) and the standard deviation of the numerator of \(H\).

Return type

tuple(float, float)

Example

from structure_factor.data import load_data
from structure_factor.spatial_windows import BoxWindow
from structure_factor.structure_factor import StructureFactor
import structure_factor.utils as utils

# load Ginibre PointPattern
ginibre_pp = load_data.load_ginibre()

# Approximating the structure factor
# creat box window
L = 70  # sidelength of the window
bounds = [[-L / 2, L / 2], [-L / 2, L / 2]]  # bounds of the window
window = BoxWindow(bounds)  # create a cubic window
# restrict to box window
ginibre_pp_box = ginibre_pp.restrict_to_window(window)
# scattering intensity
sf_ginibre_box = StructureFactor(ginibre_pp_box)  # initialize the class StructureFactor
k_norm, si = sf_ginibre_box.scattering_intensity(k_max=6, meshgrid_shape=(200, 200))


# test effective hyperuniformity
from structure_factor.hyperuniformity import Hyperuniformity

# initialize Hyperuniformity
hyperuniformity_test = Hyperuniformity(k_norm, si)
# regularization of the approximated result
hyperuniformity_test.bin_data(bins=40)
# find the H index
H_ginibre, std = hyperuniformity_test.effective_hyperuniformity(k_norm_stop=4)
print("H_ginibre=", H_ginibre)

Note

A stationary isotropic point process \(\mathcal{X} \subset \mathbb{R}^d\), is said to be effectively hyperuniform if \(H \leq 10^{-3}\) where

\[H = \frac{\hat{S}(\mathbf{0})}{S(\mathbf{k}_{peak})}\cdot\]

  • \(S\) is the structure factor of \(\mathcal{X}\),

  • \(\hat{S}(\mathbf{0})\) is a linear extrapolation of the structure factor at \(\mathbf{k}=\mathbf{0}\),

  • \(\mathbf{k}_{peak}\) is the location of the first dominant peak value of \(S\).

See [Tor18] (Section 11.1.6) and [KLovriAC+19] (supplementary Section 8).

Important

To compute the numerator \(\hat{S}(\mathbf{0})\) of the index \(H\), we fit a line using a linear regression with least square fit of the approximated structure factor sf associated to the values in k_norm until the threshhold k_norm_stop (which must be chosen before the stabilization of the structure factor around 1). If the standard deviations of sf are provided in the attribute std_sf then these values will be considered while fitting the line.

hyperuniformity_class(k_norm_stop=1, **kwargs)[source]¶

Fit a polynomial \(y = c \cdot x^{\alpha}\) to the attribute sf. \(\alpha\) is used to specify the possible class of hyperuniformity of the associated point process.

Parameters

k_norm_stop (float, optional) – threshhold on sf used for fitting the polynomial. Defaults to None.

Keyword Arguments

kwargs (dict) –

keyword arguments (except "sigma") of scipy.scipy.optimize.curve_fit parameters.

Returns

The power decay of the structure factor and the associated approximated \(S(0)\).

Example

import numpy as np

from structure_factor.data import load_data
from structure_factor.spatial_windows import BoxWindow
from structure_factor.structure_factor import StructureFactor

# load Ginibre PointPattern
ginibre_pp = load_data.load_ginibre()

# Approximating the structure factor
# creat box window
L = 70  # sidelength of the window
bounds = np.array([[-L / 2, L / 2], [-L / 2, L / 2]])  # bounds of the window
window = BoxWindow(bounds)  # create a cubic window
# restrict to box window
ginibre_pp_box = ginibre_pp.restrict_to_window(window)
# scattering intensity
sf_ginibre_box = StructureFactor(ginibre_pp_box)  # initialize the class StructureFactor
norm_k, si = sf_ginibre_box.scattering_intensity(k_component_max=6, meshgrid_shape=200)


# test hyperuniformity class
from structure_factor.hyperuniformity import Hyperuniformity

# initialize Hyperuniformity
hyperuniformity_test = Hyperuniformity(norm_k, si)
# regularization of the approximated result
hyperuniformity_test.bin_data(bins=40)
# find power decay
sf_power_decay, c = hyperuniformity_test.hyperuniformity_class(k_norm_stop=1)
print(
    "The estimated power of the decay to zero of the approximated structure factor is:",
    sf_power_decay,
)

The estimated power of the decay to zero of the approximated structure factor is: 1.93893628269006

Note

For a stationary hyperuniform point process \(\mathcal{X} \subset \mathbb{R}^d\) such that \(\vert S(\mathbf{k})\vert\sim c \Vert \mathbf{k} \Vert^\alpha\) in the neighborhood of 0, we have([Cos21], Section 4.1)

  • If \(\alpha > 1\), then \(Var\left [\mathcal{X}(B(0,R))\right ] = O(R^{d-1})\), correpsonding to the class 1 of hyperuniformity.

  • If \(\alpha =1\), then \(Var\left [\mathcal{X}(B(0,R))\right ] = O(R^{d-1}\log(R))\), correpsonding to the class 2 of hyperuniformity.

  • If \(\alpha \in ]0,1[\), then \(Var \left [\mathcal{X}(B(0,R))\right ] = O(R^{d-\alpha})\), correpsonding to the class 3 of hyperuniformity.
