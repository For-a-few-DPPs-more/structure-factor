Hyperuniformity¶

class structure_factor.hyperuniformity.Hyperuniformity(k_norm, sf, std_sf=None)[source]¶

Bases: object

Compute indicator of hyperuniformity of a stationary isotropic (or effectively isotropic) point process \(\mathcal{X} \subset \mathbb{R}^d\), given its structure factor.

Parameters

  • k_norm (numpy.array) – Vector of wavenumbers (i.e. norms of the wave vectors).

  • sf (numpy.array) – Evaluations of the structure factor, of the given point process, at k_norm.

  • std (np.array, optional) – Standard deviations associated to sf. Defaults to None.

Definition

A stationary point process \(\mathcal{X}\) is said to be hyperunifrom if its structure factor \(S\), vanishes at 0.

Note

This class contains:

  • bin_data(): Method for regularizing sf.

  • effective_hyperuniformity(): Test of effective hyperuniformity [].

  • hyperuniformity_class(): Test of the possible class of hyperuniformity [Cos21].

Typical usage:

  1. Estimate the structure factor of a point process by one of the methods of StructureFactor.

  2. Regularize the results using bin_data().

  3. Test the effective hyperuniformity using effective_hyperuniformity().

  4. If the results of effective_hyperuniformity() approve the effective hyperuniformity hypothesis, find the possible class of hyperuniformity using hyperuniformity_class().

Definition

A stationary isotropic point process \(\mathcal{X} \subset \mathbb{R}^d\), is said to be effectively hyperuniform if \(H \leq 10^{-3}\) where

\[H = \frac{\hat{S}(\mathbf{0})}{S(\mathbf{k}_{peak})}\cdot\]

  • \(S\) is the structure factor of \(\mathcal{X}\),

  • \(\hat{S}(\mathbf{0})\) is a linear extrapolation of the structure factor at \(\mathbf{k}=\mathbf{0}\),

  • \(\mathbf{k}_{peak}\) is the location of the first dominant peak value of \(S\).

See [Tor18] (Section 11.1.6) and [] (supplementary Section 8).

Important

To compute the numerator \(\hat{S}(\mathbf{0})\) of \(H\), a line is fitted using a linear extrapolation with a least-square fit on the values of sf associated to the sub-vector of k_norm truncated around the threshold k_norm_stop. k_norm_stop must satisfy a good compromise of being close to zero but also allowing to fit the line on sufficient number of points. If the standard deviations of sf are provided in the attribute std_sf then these values will be considered while fitting the line.

hyperuniformity_class(k_norm_stop=1, **kwargs)[source]¶

Fit a polynomial \(y = c \cdot x^{\alpha}\) to the attribute sf around zero. \(\alpha\) is used to specify the possible class of hyperuniformity of the associated point process (as described bellow).

Parameters

k_norm_stop (float, optional) – Threshold on sf used for fitting the polynomial. The polynomial will be fitted on the values of sf going from zero until the value associated to k_norm_stop. Defaults to None.

Keyword Arguments

Definition

For a stationary hyperuniform point process \(\mathcal{X} \subset \mathbb{R}^d\) such that \(\vert S(\mathbf{k})\vert\sim c \Vert \mathbf{k} \Vert^\alpha\) in the neighborhood of 0, see [Cos21] (Section 4.1.) we have,

Class

\(\alpha\)

\(\mathbb{V}\text{ar}\left[\mathcal{X}(B(0, R)) \right]\)

I

\(> 1\)

\(\mathcal{O}(R^{d-1})\)

II

\(= 1\)

\(\mathcal{O}(R^{d-1}\log(R))\)

III

\(]0, 1[\)

\(\mathcal{O}(R^{d-\alpha})\)
